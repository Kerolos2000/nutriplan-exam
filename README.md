## ‚ú® Key Features

- **Recipe Discovery**: Browse a wide variety of meals categorized by cuisine and category.
- **Detailed Nutritional Data**: Get precise information on calories, macronutrients (proteins, fats, carbs), vitamins, and minerals for every meal.
- **Product Scanner**: View nutrition facts for various food products with an intuitive interface.
- **Food Logging**: Track your daily meals and monitor your nutritional intake over time.
- **Interactive UI**: A premium, "frosted glass" dashboard built for a seamless user experience.
- **Multi-language Support**: Fully optimized for English and Arabic (RTL) layouts.
